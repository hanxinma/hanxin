Han Heart Kombat คืออะไร? นิยมพูดหนึ่งคำสูงสุด 4 ปุ่มสามารถพิมพ์ได้ หนึ่งคำโดยทั่วไป 2 ปุ่มสามารถพิมพ์ได้คำทั่วไป 3 ปุ่มตรงออก คล้ายกับห้าปากกาการพิมพ์ไม่จำเป็นต้องเลือกและปุ่มกดน้อย
อย่างเช่นอยากตี xiang การสะกดคู่แค่ตี xd ก็แปลว่า xiang ซึ่งก็คือคําหนึ่งคําพิมพ์ได้สูงสุดสองตัว นี่ก็คือการสะกดคู่ เหมาะกับการเล่นตาบอดหลังแบบคุ้นเคย ตอนแรกไม่สุก จะใช้ไม่ชิน เมื่อสุกใช้แล้วจะพึ่งพา
Hanxin Kombat Little Wolf รุ่นมิลลิลิตรสามารถรองรับการเพิ่มรหัสเสริมในประโยคยาวโดยไม่ต้องกดปุ่มเว้นวรรคฟังก์ชั่นเสริมที่สมบูรณ์แบบมากขึ้น สะดวกสบายและเข้ากันได้อย่างรวดเร็ว

Hanxin พิมพ์ฉบับเต็มอย่างรวดเร็ว คําเดียวสามารถเสริมได้ และยังสามารถพิมพ์คํายาวด้วยรหัสย่อได้ รหัสย่อของผ้าม่าน chuanglian พิมพ์ clg, g เป็นตัวอักษรตัวแรกของคําพ้องเสียง gai ของคําฝา

สรุปก็คือการใช้คู่สะกด 2 คีย์หนึ่งคำสั้นความยาวรหัส และการใช้ชิ้นส่วนง่ายไปที่น้ำหนักเพื่อให้บรรลุการพิมพ์โดยไม่ต้องพลิกหน้าหาคำวิธีการพิมพ์อย่างรวดเร็ว แน่นอน Hanxin มีหลายรุ่นส่วนใหญ่มีรุ่นแขวนเสริมและรุ่นพิมพ์ด่วนและคำประโยครุ่น
ฉบับพิมพ์ด่วนเป็นคำประโยคทั้งหมด 4 ปุ่ม
รุ่นคําและประโยค เป็นคําและประโยคที่หลังสามารถเพิ่มรหัสเสริมได้ ไม่ต้องกดช่องว่างเพื่อพิมพ์โดยตรง
แขวนเวอร์ชันเสริมส่วนใหญ่แขวนอยู่บนวิธีการป้อนข้อมูลฝ่ามือ Sogou

หัวใจฮั่นเป็นเหมือนครอบครัวมากกว่า เข้าใจในสิ่งที่คุณคิด และรักษาความรู้สึกปลอดภัยของคุณในทุกด้าน
ฮั่นซินทําเว็บไซต์มาหลายสิบปีหรือหลายร้อยปี ไม่มีใครดูแล เว็บไซต์และทรัพยากรยังคงอยู่ ดังนั้นคุณจะสามารถหารหัสหัวใจฮั่นได้ ไม่ต้องกังวลว่าจะเปลี่ยนคอมพิวเตอร์แล้วจะไม่พบฮาร์ทโค้ดหรือซอฟต์แวร์แล้ว
ฮั่นซินมีเว็บไซต์แล้วไม่ต้องต่ออายุ ดังนั้นไม่ต้องกังวลว่าจะไม่มีใครต่ออายุเว็บไซต์แล้วหายไป แล้วหากบริษัทแม่ของเว็บไซต์นี้ไม่อยู่แล้ว ใจหานยังมีตัวสำรองอีก 1 สถานี ใช้ได้ แล้วหากสำรอง 1 สถานี บริษัทหลักไม่อยู่แล้ว นอกจากนี้ยังมีสถานีสํารอง 2 สถานีที่สามารถใช้ได้ นี่เป็นสถานีของ Tencent หากเว็บไซต์นี้ยังคงหายไปก็ยังมี 3 สถานีสำรองที่จะใช้ การสำรองข้อมูล 3 สถานีเป็นคลังสินค้ารหัสโปรแกรมเมอร์ที่ใหญ่ที่สุดในโลก ไม่ล้มง่ายๆ ทรัพยากรเหล่านี้หาที่ไหน แนะนําให้ค้นหาด้วย bing.com ค้นหาได้ง่ายขึ้น
ซอฟต์แวร์ใช้ซอฟต์แวร์โอเพนซอร์สและโดยทั่วไปจะมีคนอัปเดตเสมอ จากนั้นซอร์สโค้ด Hanxin ก็เปิดเผยเช่นกัน ทุกคนสามารถเพิ่มลงในซอฟต์แวร์โอเพนซอร์สเพื่อสร้างวิธีการป้อนข้อมูล Hanxin ได้ ในที่สุด Hanxin จะทําการบํารุงรักษาส่วนตัวเป็นบทเรียน เพื่อให้ทุกคนสามารถบำรุงรักษาและอัปเดตได้อย่างง่ายดาย
กล่าวโดยสรุป Hanheart Code Open Source (ทุกคนสามารถนําเข้าวิธีการป้อนข้อมูลใหม่มาใช้ได้) และวิธีการป้อนข้อมูลยังใช้วิธีการป้อนข้อมูลแบบโอเพนซอร์ส (ทุกคนสามารถรักษาซอฟต์แวร์ได้) ดังนั้น Hanxin จึงสามารถปกป้องความรู้สึกปลอดภัยของผู้ใช้งานได้ในทุกด้าน
